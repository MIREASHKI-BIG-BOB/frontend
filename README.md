# стартовая страница

Стек: Vite + React + TypeScript, TailwindCSS, Ant Design v5, Recharts.

## Быстрый старт (Windows PowerShell)

```powershell
# Установка зависимостей
npm install

# Запуск дев-сервера
npm run dev

# Сборка
npm run build

# Предпросмотр сборки
npm run preview
```
